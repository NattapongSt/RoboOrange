# RoboOrange

##Setup

```bash
git clone https://github.com/NattapongSt/RoboOrange
```

##Change Directory to RoboOrange

```bash
cd ./RoboOrange
```

##Create python virtual environment

```bash
python -m venv venv
```

##Activate virtual environment

```bash
source venv/bin/activate
```

##Install library

```bash
pip install -r requirements.txt
```

##Run systems

```bash
python GUI.py
```
